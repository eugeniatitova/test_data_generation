# test_data_generation

### Требования
- Python == 3.6
- PyTorch == 1.6.0
- Pytorch Lightning == 0.10.0 

### Установка зависимостей
```
$ pip install -r requirements.txt
```

### Использование
```
$ python main.py -c configs/<config-file-name.yaml>
```

### Просмотр логов

```
$ tensorboard --logdir ./tensorboard_logs
```
